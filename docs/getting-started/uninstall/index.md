## Uninstall the `shopify-cli` package

### Ruby gem

```console
$ gem uninstall shopify-cli
```

### Brew (MacOS)

```console
$ brew uninstall shopify-cli
```

### apt (Debian, Ubuntu)

```console
$ apt uninstall shopify-cli
```

### RPM (CentOS, Fedora, RHEL, SUSE)

```console
$ yum remove shopify-cli
```
